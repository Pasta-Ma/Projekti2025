# MeetMe
Kouluprojekti


Käyttöön otto:
Frontend
  1:  cd frontend
  2:  npm install
  
Backend
  1: cd backend
  2: npm install


  SQL Database löytyy rootista = db_dump.dump


  Projektin startti:
    1: cd backend 
    2: npm run dev
    3: cd frontend
    4: npm start
    5: nettisivu aukeaa localhost:3000

  Nettisivun käyttö:
    1: Sign up: Luo käyttäjä.
    2: Sign in: Kirjaudu luodulla käyttäjällä
    (3: Sivulla on Admin käyttäjiä joilla on enemmän oikeuksia, niihin pääsee kirjautumalla: 
    Sähköposti = admin@outlook.com
    Salasana = admin)

  Tekoälyrapotti:
    Projektissa hyödynsin Copilot:
      1: Middleware - adminMiddleware.js on kokonaan copilotin tuottama.
      2: .css - Käytin näissä Copilotin apua hienosäännössä.
      3: Register.js - Tässä käytin Copilot että sain automaattisen siirron Login kohtaan toimimaan.
      4: TestApi.js - On Copilotin luoma testi tiedosto, jota käytin backend databasea testatessa.
      5: Käytin Copilot kirjoitus virheissä ja errorien selventämisessä.
